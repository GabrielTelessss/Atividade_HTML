Ferramentas Utilizadas

- voltar

O HTML (HyperText Markup Language) é a linguagem de marcação utilizada para criar páginas da web.
Não é uma linguagem de programação, mas sim uma linguagem de marcação que descreve a estrutura e o conteúdo de uma página da web.
Existem várias tags e elementos HTML que podem ser usados para criar e formatar o conteúdo da página.
Abaixo, você encontrará um resumo das principais ferramentas e elementos HTML

Listas Ordenadas e Não Ordenadas

- voltar

As listas ordenadas são usadas quando você deseja criar uma lista de itens que deve seguir uma ordem específica.
Os itens em uma lista ordenada são automaticamente numerados ou marcados com números, letras ou outro tipo de contador.
Para criar uma lista ordenada, você usa a tag ol (que significa "ordered list") e a tag li (que significa "list item") para cada item da lista

As listas não ordenadas são usadas quando você deseja criar uma lista de itens que não segue uma ordem específica e não é numerada.
Os itens em uma lista não ordenada são geralmente marcados com símbolos de marcadores, como pontos ou círculos.

Links

- voltar

Em HTML, você pode criar links para direcionar os visitantes do seu site para outras páginas da web, recursos, arquivos, ou até mesmo dentro da mesma página.
A tag principal para criar links é , que significa "anchor" (âncora).

Você pode controlar como o link é aberto usando o atributo target. Os valores comuns são _blank para abrir o link em uma nova janela ou guia e _self para abrir na mesma janela.
